# SHIFT Fitness Prototype

A mobile-first React/Vite prototype based on the SHIFT app flow and visual identity.

## Fastest way to run it in Visual Studio Code

1. Unzip the folder.
2. Open the **SHIFT_Website_Ready** folder in Visual Studio Code.
3. Open **Terminal → Run Task**.
4. Choose **SHIFT: Install & Start**.
5. Vite opens the prototype in your browser.

You can also run manually:

```bash
npm install
npm run dev
```

## Main user flow

- **Sign up** → questionnaire 1 → questionnaire 2 → questionnaire 3 → account creation → dashboard
- **Log in** → login form → dashboard
- **Do it later** → dashboard as guest

## Reference images

The original supplied screens are stored in `/references` so you can compare the coded prototype with the visual direction.

## Replace mascot images

Final mascot assets are in `/src/assets`. You can replace any PNG while keeping the same filename.

## Reset the prototype

Open **Profile** inside the app and click **Reset prototype**. This clears local storage and returns to the entry screen.
